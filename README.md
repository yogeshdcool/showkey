
# Showkeys

A tool to display keybindings/hotkeys


## Installation

```bash
  git clone https://github.com/yogeshdcool/showkeys.git
  cd showkeys
  python showkeys.py
```
    
## Features

- Supports rofi and dmenu
- Window manager,desktop environment independent
- Easily extensible
